# \# 📚 Hindi GATE Tutor (RAG-based Learning Assistant)

# 

# An AI-powered tutoring system that allows students to upload PDFs and ask questions in a conversational manner.

# 

# \---

# 

# \## 🚀 Features

# 

# \- 📄 Upload and ingest PDFs

# \- 🔍 Hybrid Retrieval (Dense + BM25)

# \- 🎯 Cross-Encoder Reranking

# \- 💬 Conversational Memory (follow-up questions supported)

# \- 🔄 Query Rewriting for better context understanding

# \- 📚 Document Library with per-file deletion

# \- 🖥️ Streamlit Chat UI

# 

# \---

# 

# \## 🧠 Tech Stack

# 

# \- FastAPI (Backend)

# \- Streamlit (Frontend UI)

# \- Qdrant (Vector Database)

# \- Sentence Transformers (Embeddings)

# \- Cross-Encoder (Reranking)

# \- BM25 (Keyword Search)

#
